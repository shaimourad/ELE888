This repository contains projects, simulations, and coursework completed for ELE888 at Toronto Metropolitan University. The course focused on intelligent systems, computational methods, and the application of artificial intelligence concepts in engineering systems.

Topics covered include:

Intelligent and adaptive systems
Machine learning fundamentals
Neural networks
Fuzzy logic systems
Pattern recognition
Data-driven engineering applications

Projects emphasized algorithm development, system modeling, simulation, and applying intelligent decision-making techniques to real-world engineering problems.
